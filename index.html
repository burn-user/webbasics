Для чего нужен язык HTML?

HTML - язык гипертекстовой разметки. Основная задача данного языка заключается в создании каркаса сайта и определении расположения элементов сайта на экране.

Структура HTML-файла

<!DOCTYPE html>
<html>
  <head>
    (служебная информация)
  </head>
  <body>
    (содержимое сайта)
  </body>
</html>
Язык HTML состоит из единиц, называемых тегами. Это специальные команды, выполняющие определённую функцию.
Все теги можно разделить на 2 вида: парные и непарные. Отличить их довольно просто: непарные теги состоят из одного слова-команды, а парные - из двух. Например, <br> - непарный тег, а <h1></h1> - парный.

Текст и шрифты

Рассмотрим наиболее популярные теги, позволяющие изменять текст.
<h1></h1> - заголовки. Данный тег имеет 6 уровней: h1, h2, h3, h4, h5 и h6. Уровень h1 - самый главный, а h6 - минимальный. С их помощью можно создавать иерархию в тексте, распределяя заголовки текста по разным уровням.
<br> - перенос строки. Переносит курсор на новую строчку, имитируя нажатие Enter.
<p></p> - абзац.
<b></b> - жирный шрифт.
<strong></strong> - жирный шрифт + относится к элементам логической разметки, указывая браузеру на важность текста.
<i></i> - курсив.
<u></u> - подчёркнутый текст.
<strike></strike> - перечёркнутый текст.
<sup></sup> - верхний индекс.
<sub></sub> - нижний индекс.
<a></a> - ссылка.
<a href="page.html">ссылка</a>
href - атрибут тега <a></a>. В кавычках необходимо указать ссылку на сайт или элемент страницы.
<ul></ul> - маркированный список.
<ol></ol> - нумерованный список.
<li></li> - элемент списка.
<ul>
   <li>строка 1</li>
   <li>строка 2</li>
   <li>строка 3</li>
</ul>
Стоит отметить, что данные теги (кроме последних трёх) редко используются при создании сайтов, так как их функционал удобнее реализовывать с помощью CSS.

Структурные теги

Это теги, создающие структуру сайта.
<div></div> - блок. Данный тег можно считать одним из самых популярных среди всех тегов HTML.
<span></span> - блок со свойствами строки. Также является весьма популярным в HTML.

Таблицы

<table></table> - контейнер для таблицы.
<tr></tr> - строка.
<td></td> - ячейка.
<table>
   <tr>
      <td></td>
      <td></td>
   </tr>
   <tr>
      <td></td>
      <td></td>
   </tr>
</table>

Изображение

<img src="picture.jpg"> - изображение. В атрибуте src необходимо указать название изображения.

Для чего нужен язык CSS?

CSS - каскадные таблицы стилей. Данный язык используется для стилизации веб-сайтов, а также для более точной настройки отображения элементов сайта на экране.
Принцип работы CSS крайне прост - обращаясь к тегам в файле HTML, вы можете прописать им какие-либо свойства (размер, цвет, границы, шрифты, анимации и т. д.), причём применять их можно в абсолютно любых комбинациях.

Подключение к файлу HTML

Стили CSS применяются к тегам HTML, поэтому CSS без HTML существовать не может.
CSS можно подключить к HTML двумя способами:
  • Интегрировать CSS непосредственно в код HTML
  • Подключить отдельный файл CSS к файлу HTML
Чтобы интегрировать CSS в код HTML, необходимо использовать тег <style></style>:
<body>
   <style> (код CSS) </style>
</body>
Подключить отдельный файл CSS к файлу HTML можно с помощью тега <link>:
<head>
   <link rel="stylesheet" href="style.css">
</head>

Как применять свойства CSS к тегам HTML

Первым шагом вам необходимо присвоить тегу класс:
<div class="abc"></div>
Затем в коде CSS нужно записать следующую конструкцию:
.abc {
   (здесь прописываются какие-либо свойства)
}

Наиболее популярные свойства CSS

width и height задают блоку ширину и высоту соответственно. Например: width: 150px; (ширина 150 пикселей).
margin - внешний отступ от блока. Например: margin: 10px; (отступ 10 пикселей).
padding - внутренний отступ блока.
color - изменение цвета текста блока. Для записи данного свойства можно использовать rgb(): color: rgb(255, 80, 124);. Для некоторых цветов (белый, чёрный, синий, красный и т. д.) можно использовать запись одним словом. Например: color: white;.
background-color - изменение цвета блока.
border - граница блока. Например: border: 2px solid green (граница толщиной 2 пикселя, непрерывная, зелёная).

Вам понадобятся всего лишь две программы: Блокнот (встроенная утилита в ОС Windows 10) и любой браузер.
